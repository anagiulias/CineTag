# CineTag

## Comandos
Para rodar esse projeto, você precisa utilizar os seguintes comandos:

### `npm install`
### `npm start`

**Observação: Os comandos precisam da instalação do NodeJS em sua máquina, eles podem ser feitos no seu terminal.**

## Tecnologias utilizadas:
* HTML
* CSS
* JavaScript

### Tecnologias extras:
* React
* React-router-dom
* Module CSS
* ContextAPI
* Hooks
